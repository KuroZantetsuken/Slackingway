# Slackingway

Dynamic frame rate limiter to allow GPU headroom for Windows to do other things, preventing other apps like Discord and YouTube from freezing when FFXIV is in focus.

## Installation

Add the following Custom Repository URL to your Dalamud Plugin Installer settings:
```
https://raw.githubusercontent.com/KuroZantetsuken/Slackingway/main/pluginmaster.json
```
